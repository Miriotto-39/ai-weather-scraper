# AI Weather Scraper (長岡市お天気レポート自動生成)

## 概要
指定したWebサイトから長岡市の最新の天気・気温情報を自動的に取得（スクレイピング）し、OpenAIのGPT-4o-miniを使用して簡潔なレポートを作成。
結果を日付付きのテキストファイルとして自動保存するツールです。

## 使用技術
- Python 3.12
- FastAPI (APIサーバー)
- BeautifulSoup4 (スクレイピング)
- OpenAI API (GPT-4o-miniによるテキスト分析)

## 機能
- `/make-report` にアクセスすると実行
- Webサイトから2000文字のテキストを抽出
- AIによる情報の要約
- 実行時の日時を含めたファイル名でのレポート保存
